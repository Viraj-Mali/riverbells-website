# Riverbells Resort Website

## How to Deploy (2 minutes)

### Option 1: Netlify (Recommended — FREE)

1. Go to [app.netlify.com](https://app.netlify.com)
2. Sign up / log in with GitHub or email  
3. Click **"Add new site"** → **"Deploy manually"**
4. **Drag & drop** the entire `binary-photon` folder into the upload area
5. Done! Your site is live at a `.netlify.app` URL

**Custom domain:** Go to Site Settings → Domain Management → Add custom domain → Enter `bookriverbells.com` → Follow DNS setup instructions

---

### Option 2: Vercel (FREE)

1. Go to [vercel.com](https://vercel.com) → Sign up
2. Click **"Add New"** → **"Project"**  
3. Choose **"Import from directory"** or drag-drop
4. Deploy — live in seconds

---

### Option 3: GitHub Pages (FREE)

1. Create a GitHub repo named `riverbells-website`
2. Upload all files (`index.html`, `style.css`, `script.js`, `images/`)
3. Go to Settings → Pages → Source: `main` branch → Save
4. Site will be at `yourusername.github.io/riverbells-website`

---

## After Deployment

- **Test on mobile**: Open the URL on your phone
- **Share it**: Send the link to anyone for review
- **Custom domain**: Point `bookriverbells.com` to your hosting provider
